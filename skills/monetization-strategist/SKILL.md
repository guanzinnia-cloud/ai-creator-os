## 渐进式变现路径（Agent Economics 版）

### 阶段1：0-1K粉丝（免费层 → 数据验证）
- **目标**：验证内容-用户-变现匹配度，积累 Ontology 数据
- **策略**：
  - 免费公开内容（建立信任、积累数据）
  - 轻量测试：工具类 affiliate（AI产品推荐，按注册/付费分成）
  - 数据收集：记录每条内容的"变现潜力评分"
- **定价**：$0（用户获取成本由内容时间覆盖）
- **里程碑**：找到3个高变现潜力方向，LTV/CAC 模型初建

### 阶段2：1K-10K粉丝（Pro 层 → 订阅+按量）
- **目标**：建立稳定收入流，验证付费意愿
- **策略**：
  - **Pro 订阅**：$9.9/月（基础 Agent 使用）或 $29.9/月（多 Agent Workflow）
  - **超额按量**：超出配额后按 Token/任务计费
  - **知识付费**：付费专栏/小册（基于知识库 Agent 输出的结构化内容）
  - **社群运营**：高质量粉丝群（飞书/Discord），付费门槛 $49
- **定价模型**：订阅+配额+API（参考 Cursor/Claude Coding Plan）
- **里程碑**：MRR 达到当地最低工资水平，LTV/CAC > 3

### 阶段3：10K+粉丝（Enterprise → 私有化+效果付费）
- **目标**：多元化收入，品牌化运营
- **策略**：
  - **品牌合作**：AI工具/科技产品测评（按粉丝数报价）
  - **私有化部署**：为企业定制 Agent 系统（本地模型 + 定制 Ontology）
  - **效果付费**：按"任务完成效果"计费（如：每月产出 30 条视频 / 互动率提升 20%）
  - **平台激励**：YouTube Partner/小红书创作者激励
  - **咨询/演讲**：AI产品方法论输出
- **定价模型**：项目制 $5K-50K + 年度维护费，或效果分成
- **里程碑**：月收入超过主业收入

## Agent Economics 核心公式

### 1. Token 成本控制
单条视频 Token 成本 = 输入 Token × 输入单价 + 输出 Token × 输出单价
月度 Token 预算 = 日均产出量 × 单条成本 × 30
降本策略：
边缘计算：Edge-TTS 替代 ElevenLabs（语音成本降 90%）
本地渲染：Remotion 替代 Runway（视频成本降 100%）
缓存复用：知识库检索结果缓存，减少重复调用

### 2. LTV/CAC 计算
CAC = 营销费用 + 获客时间成本 / 新增付费用户数
LTV = ARPU × 平均付费周期 × 毛利率
健康标准：LTV/CAC > 3
警戒线：LTV/CAC < 1.5（立即停止投放）

### 3. 效果付费模型
计费基础 = 任务完成量（视频数/报告数/分析数）
单价 = 基础效果价 × 质量系数（完播率/互动率/转化率）
而非：按 Token 消耗量计费（用户不可预测）
