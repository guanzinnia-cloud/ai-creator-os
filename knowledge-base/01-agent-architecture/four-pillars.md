# Andrew Ng 四支柱框架

## 记忆口诀：Red Turtles Paint Murals

### 1. Reflection（反思）
- AI 自我审查与迭代改进
- 在本项目中的映射：每个 Skill 的 self_improving 段落，视频数据回流自动修正策略

### 2. Tool Use（工具使用）
- 扩展 LLM 能力至外部工具
- 在本项目中的映射：WebFetch（热点抓取）、飞书 API（知识库读写）、Git（版本控制）

### 3. Planning（规划）
- 复杂多步任务规划
- 在本项目中的映射：内容生产流水线（热点→选题→脚本→视频→运营→变现）

### 4. Multi-Agent（多智能体）
- 专业化分工优于单一通用模型
- 在本项目中的映射：6 个 Agent 的 Hybrid 拓扑（Sequential + Parallel + Hierarchical）
