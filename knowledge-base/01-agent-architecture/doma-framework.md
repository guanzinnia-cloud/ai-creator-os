# DOMA 决策智能架构（中科闻歌）

## 来源
中科闻歌董事长王磊，"AI闪耀中国2026"对谈，人民政协网。
核心判断：AI的终局不是生成，是决策。

## 四层架构

### D - Data（数据）
- 企业/个人分散在流程和经验中的原始数据
- 在本项目中的映射：热点抓取数据、用户行为数据、视频互动数据

### O - Ontology（本体）⭐ 最关键
- 业务逻辑的结构化表达
- 王磊原话："没有本体，大模型就不懂业务"
- 在本项目中的映射：
  - 博主定位体系（AI科技+跨境电商+本地生活+好奇心+审美+商业化）
  - 用户画像本体（细分人群、痛点、场景、决策链）
  - 内容分类本体（标签体系、平台规则、风格DNA）

### M - Models（模型）
- 理解数据、参与判断的 AI 模型
- 在本项目中的映射：Claude 3.7 Sonnet（决策中枢）、Kimi（长文本解析）

### A - Agents（智能体）
- 将分析变成动作的执行层
- 在本项目中的映射：6 个 Self-improving Agent

## 核心方法论
先锚定最复杂的决策问题（内容创业的全流程），再反推 AI 需要具备的能力、系统和产品形态。
不是"接个模型试试"，而是"定义清楚业务本体，再让模型和 Agent 在结构化规则里运行"。
