**面试陷阱**：上下文Token也计入输入。一个RAG Agent单次查询的实际成本 = 用户问题Token + 检索出的文档片段Token + 系统提示词Token，后两项往往占输入的80%以上。

## 二、Agent 单次任务成本拆解

### 2.1 RAG Agent 成本拆解公式

> **总成本 = 用户查询Token成本 + 检索上下文Token成本 + 生成回答Token成本 + Embedding检索成本 + Reranker重排成本 + 插件调用成本**

**场景假设**：企业员工问"今年的年假政策是什么？"
- 用户查询：~75 token
- 检索5条文档片段：~2,250 token
- Agent生成回答：~300 token

**逐步计算**（DeepSeek-V3 + ADP定价）：

| 成本项 | Token数 | 单价 | 单次成本 | 说明 |
|--------|---------|------|---------|------|
| 用户查询（输入） | 75 | ¥0.002/千token | ¥0.00015 | 问题本身 |
| 检索文档（输入上下文） | 2,250 | ¥0.002/千token | ¥0.0045 | 5条×450token |
| 生成回答（输出） | 300 | ¥0.008/千token | ¥0.0024 | 回答文本 |
| Embedding检索 | 2,250 | ¥0.0005/千token | ¥0.001125 | youtu-embedding |
| Reranker重排 | 2,250 | ¥0.0005/千token | ¥0.001125 | youtu-reranker |
| **小计（纯RAG）** | — | — | **¥0.0093** | **约¥93/万次** |
| 联网搜索插件（可选） | — | ¥0.022/次 | ¥0.022 | 每次调用固定价 |
| **总计（含联网）** | — | — | **¥0.0313** | **约¥313/万次** |

### 2.2 不同 Agent 类型成本对比

| Agent类型 | 单次Token消耗 | 单次成本(ADP) | 适用场景 |
|-----------|-------------|-------------|---------|
| **简单RAG问答** | ~3,000 | ¥0.009 | 知识库查询 |

## 三、SaaS 商业化核心指标体系

### 3.1 三层指标体系与 2026 年行业基准

| 层级 | 核心指标 | 行业基准 | 计算公式 |
|------|---------|---------|---------|
| **收入层** | NRR（净收入留存率） | **≥110%健康，≥125%世界级** | (期初MRR + 增购 - 缩减 - 流失) / 期初MRR |
| | 毛利率 | **SaaS≥70%为健康线** | (收入 - COGS) / 收入 |
| | 40法则 | **ARR增长率% + 利润率% ≥ 40%** | SaaS效率核心指标 |
| **获客层** | LTV/CAC | **≥3:1健康，≥5:1优秀** | LTV / CAC |
| | CAC回收期 | **<18个月健康** | CAC / (ARPA × 毛利率) |
| **客户层** | 续约率 | **≥90%为健康线** | 续约客户数 / 应续约客户数 |
| | Onboarding周期 | **14-30天** | 签约到首次价值实现的时间 |

### 3.2 LTV 完整计算

> **LTV = ARPA × 毛利率 / 客户流失率**

**计算案例**：
- 月订阅¥298/席位，客户平均购买10个席位 → 月ARPA = ¥2,980
- 毛利率 = 75%
- 月流失率 = 5%
- **LTV = ¥2,980 × 75% × 20个月 = ¥44,700**
- **目标CAC < ¥14,900**（LTV/CAC = 3:1）

### 3.3 盈亏平衡点

> **盈亏平衡点（月调用次数）= 固定成本 /（单次定价 - 单次变动成本）**

**案例**：企业知识库Agent产品
- 固定成本：¥10,000/月
- 变动成本：¥0.01/次
- 定价：¥0.05/次
- **盈亏平衡 = ¥10,000 / (¥0.05 - ¥0.01) = 250,000次/月**

## 五、云产品计费模型

| 计费模式 | 适用场景 | 北京地域单价 | 折扣空间 |

| **包年包月** | 长期稳定业务 | 内存¥37/GB/月 + 磁盘¥1/GB/月 | **相当于按量计费的37-40折** |

- 包年包月：**¥1,284/月**（省31%）
- 预留实例：**¥899/月**（省52%）

## 六、成本核算速查公式表

| 公式名称 | 公式 | 变量说明 |
|---------|------|---------|
| **Token调用成本** | 成本 = 输入Token×输入单价 + 输出Token×输出单价 | 单价单位：元/千token或元/百万token |
| **Agent单次成本** | 总成本 = 查询Token + 上下文Token + 输出Token + Embedding + Reranker + 插件 | 各组件分别计算后求和 |
| **LTV（简化版）** | LTV = ARPA × 毛利率 / 流失率 | ARPA=每客户平均月收入 |
| **LTV/CAC比率** | LTV/CAC = LTV / CAC | 健康线≥3:1 |
| **CAC回收期** | 回收期 = CAC / (ARPA × 毛利率) | 健康线<18个月 |
| **NRR** | NRR = (期初ARR + 增购 - 缩减 - 流失) / 期初ARR | 健康线≥110% |
| **40法则** | 40法则 = ARR增长率% + 利润率% | 底线≥40% |
| **盈亏平衡点** | 平衡量 = 固定成本 / (单价 - 变动成本) | 用于计算最少客户数/调用量 |
| **毛利率** | 毛利率 = (收入 - COGS) / 收入 | SaaS健康线≥70% |
| **云产品包年包月折扣** | 折扣率 ≈ 37-40%（相对按量计费） | 预留实例再降20-30% |

> **核心原则**：
> 1. 所有定价决策必须基于成本结构（Token+云+人力）的量化拆解
> 2. 定价锚点不是成本，而是**客户感知价值**
> 3. 毛利率≥70% 是 SaaS 健康的底线，LTV/CAC≥3 是获客效率的底线
